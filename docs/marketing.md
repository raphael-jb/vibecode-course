## 1. Marketing-Produkte (Ideen & Konzepte)

### 1.1 Peer Group Session (von Raphael geleitet) — IDEE, nicht beschlossen

**Status:** Konzept-Phase, 2026-05-11.

**Idee:** Eine von Raphael geleitete Gruppensession als günstiger, niederschwelliger Einstieg in die Markenwelt. Mehrere Führungskräfte in einem Raum, Raphael als Leiter — kein 1:1.

**Funktion (warum es zur Marke passt):**
- Niedrige Eintrittsbarriere für Interessierte, die das Format erst kennenlernen wollen, bevor sie ein 1:1-Mandat erwägen.
- Sichtbarkeit für Raphaels Denkweise und Tonalität — Peer Sparring "in Aktion", aber im Format, in dem es skaliert.
- Marketing-Vehikel, kein Core-Produkt. Klar als Einstieg, nicht als Ersatz für 1:1-Sparring kommuniziert.

**Wichtige Abgrenzung:** Dies ist **nicht** "Peer Sparring" im Sinne von Abschnitt 1.1 in products.md. Peer Sparring bleibt definitionsgemäß 1:1. Die Gruppensession braucht einen anderen Namen, damit die Kategorie sauber bleibt.

**Offene Fragen:**
- [ ] Wie heißt das Format? (Arbeitstitel, klar von "Peer Sparring" unterscheidbar)
- [ ] Gruppengröße? (Quorum, das Tiefe noch erlaubt — vermutlich 4–8)
- [ ] Frequenz: einmalig, regelmäßig oder Mini-Serie?
- [ ] Preisanker: was ist "günstig genug" als Einstieg, aber nicht so günstig, dass es das Core-Produkt entwertet?
- [ ] Inhaltliche Form: thematisch geclustert, fallbasiert, offen?
- [ ] Wo in der User Journey sitzt das? (Eigene Page? Sub-Sektion auf `/sparring`? Pop-up-Format?)
